# Development Scripts

This directory contains cross-platform development scripts for the Hyperloop H10 project.

## Quick Reference

```bash
# Unix/Linux/macOS
./scripts/dev.sh setup
./scripts/dev.sh all

# Windows PowerShell
.\scripts\dev.ps1 setup
.\scripts\dev.ps1 all

# Windows Command Prompt
scripts\dev.cmd setup
scripts\dev.cmd all
```
